Import Menu

Virtual Pinball Backup Manager - managing your backups, one game at a time

  Restore From Backup Menu Option

The Restore From Backup menu option provides the user to re-install a game previously deleted, but for which a backup is maintained in the system.

Refer to Restore Deleted Games for details on how to restore previously deleted games from stored backups.

  Import New Menu Option

The Import New menu option provides the user the ability to install new games not currently held in the system. This can also be done via drag and drop into the Installed Games panel.

Refer to Import Games for details on how to import new games from outside of the stored backups.
