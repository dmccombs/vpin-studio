  VPBM Change Log

Note this change log takes effect from version 1.3 when VPBM became publicly available

Version 1.6
  • Added initial command line mode
  • Added ability to manuially set various game code values
  • Added support for other game codes
    • Version 1.5
    Fixes to address :
    1. Better validation of date/time fields (DateAdded, DateUpdated, and DateFileUpdated) and backup/restore these values
    2. Allow VPBM to launch without Pinup Popper database located and let use fix issue using Settings form
    3. Reordered change log entries
    Version 1.4
    Fixes identified from public release:
    1. Corrected filter searching for Pup Packs to search in PUPVideos rather than PUPVideo
    2. Added additional error checking of config file (vPinBackupManager.json) and report to user via message popups
    3. Support for backup/restore/export of games with trailing spaces
    4. Sanitise stack traces to limit paths to applicable blocks
    5. Added change log (this file)
    Version 1.3
    Initial public release, including backup encryption tied to target host
