Selected Game Details

Virtual Pinball Backup Manager - managing your backups, one game at a time

  Overview

The Selected Game Details panel provides a read only view of the currently selected games details. This information mirrors the information found within the Pinup Popper Games Manager, but is a read only copy of this information.

Users wishing to modify game data seen in this panel should do this through Pinup Popper as the scope of VPBM is limited to the management of game backups rather than editing game data.

The source of information viewed in this panel is determined by the selections made in the Selected Game Display Control panel.

  1. If Disk is selected, the displayed content is taken from the currently active Pinup Popper database.
  2. If Backup is selected, the displayed content is taken from the currently games backup file.
