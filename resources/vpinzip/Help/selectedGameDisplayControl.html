Selected Game Display Control

Virtual Pinball Backup Manager - managing your backups, one game at a time

  Overview

The Selected Game Display Control panel is provided to allow users to manage what information is displayed for the selected game. The selected game itself corresponds to the most recently selected game in the Installed Games panels game list.

There are two elements that can be controlled, they are:

  1. Whether game information is to be displayed from the Current system (currently installed game - on disk) or from the Backup. The Backup option is only made available for games that have a backup file present. By default the Current option is selected.
    Selecting one of these radio buttons toggles the displayed content source.
  2. Whether the Selected Game Details panel should be displayed or not.
