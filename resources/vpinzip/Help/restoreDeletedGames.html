Restore Deleted Game Backups

Virtual Pinball Backup Manager - managing your backups, one game at a time

  Overview

The Restore Deleted Game Backups allows backup games that are do not have a corresponding entry in the Pinup System database to be restored to the system.

This function is very similar to the Restore Selected functionality found on the Installed Games panel, which the key difference being that Restore Selected operates on games that are already present in the Pinup System while this import functionality handles games that do not already exist in the Pinup System database but for which backups exist in the appropriate backup directory - generally these will be games previously installed, backed up, and then deleted.

After performing a restore of one or more games into the system, these games will be present in the Pinup System database and shown on the Installed Games panel.

If Pinup System is configured to use active SQL playlists, then the imported games will be included in these playlists on next run. Refer to Pinup System Playlists documentation for further information on playlists. If Active SQL playlists are not being used, then users will need to manually add the installed games to Pinup System playlists using the Pinup System setup tool - refer to Playlist Config here.

Games that are deleted from the Installed Games panel are removed from the live system and from the Pinup System database, however the backup files are not deleted (if they exist) and can be restored using this dialog.


  How to Restore Deleted Game Backups

The following steps are used to import new games using menu options.

  1. Select the Restore From Backup option from the Import Menu.
  2. Select one or more games to restore from the displayed Restore Game Backups dialog.
  3. Press the Restore Selected button. The restoration of games will be triggered, with the dialog closing after all requested games are imported. Note that depending on the size of the games and their associated files, this process may be slow. Once started, an restore cannot be cancelled.

To close the dialog without performing a restore, press the Close button.

Backup files containing games with GameName or GameFileName values matching already installed games will not be available for restoration.
